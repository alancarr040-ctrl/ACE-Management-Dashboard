# Git Workflow

Typical change:

git status - Shows your dashboard

On branch main
Untracked files:
  (use "git add <file>..." to include in what will be committed)
        docs/Development/GIT.md

nothing added to commit but untracked files present (use "git add" to track)

git diff - Shows unstaged changes

git add <files> - adds a file to the commit cycle

git diff --cached - What is in the staging area

git commit -m "<message>" commits with the message 

git commit -m "Establishing new doc"

git push - Pushes to the server which git is stored (local/github)
