# Release Notes - 2.6.1 Roadmap & Vision

## Summary

2.6.1 is a documentation-only release that formalizes the identity and roadmap of **ACE Management Dashboard**.

## Highlights

- Added a public GitHub-facing roadmap.
- Added a project vision document.
- Added a cumulative changelog.
- Established **ACEMD** as the approved short name.
- Clarified that ACEMD is not ACE and is not an ACE fork.
- Explained why operational infrastructure was built before account and character administration.
- Defined the 2.x platform foundation, 3.x ACE administration, and 4.x advanced platform tracks.
- Prepared the next package definition for 2.7.0 Scheduler & Automation.

## Runtime impact

None expected. This package contains documentation and metadata updates only.
