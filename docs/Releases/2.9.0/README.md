# ACEMD 2.9.0 - Notifications & Alerts

## Summary

2.9.0 adds the ACEMD Notification Service and Notifications workspace.

This release establishes the operator-facing alert layer without adding dashboard bloat or external notification risk.

## Highlights

- Shared Notification Service.
- Notifications tab and local alert center.
- Active alert summary and navigation badge.
- Notification lifecycle actions: acknowledge and resolve.
- Health-derived alert rules.
- Metrics threshold alert rules.
- Automation warning alert rules.
- Critical event surfacing.
- Channel framework for future email/webhook integrations.
- Notification lifecycle events recorded into the Event Service.

## SQL

No SQL migration is required.
