# Release Documentation

This directory stores release-specific documentation for ACE Management Dashboard.

Each release directory may contain:

- `README.md` - package/deployment notes for that release.
- `RELEASE_NOTES.md` - release-specific change notes.
- `RELEASE_NOTES_LEGACY.md` - preserved older notes when a release had more than one historical release note source.

The repository root `CHANGELOG.md` remains the cumulative human-facing release history.
