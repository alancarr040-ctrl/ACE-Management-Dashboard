# Vision and Project Identity

This directory contains the long-form ACEMD vision and project identity documentation.

- `VISION.md` defines the product identity, mission, philosophy, and relationship between ACE and ACE Management Dashboard.
- `PROJECT.md` preserves the infrastructure and repository overview.

The product name is **ACE Management Dashboard**. The approved abbreviation is **ACEMD**. The term **ACE** refers to the upstream emulator/server project, not this dashboard.
