# ACEMD 3.1.0 - Account Management

Phase 3.1.0 begins the first true administration subsystem on top of the Phase 3 Discovery Framework.

## Highlights

- Read-only Account Management module.
- Account Explorer with search, sort, filter, and pagination controls.
- Summary cards for account population, bans, elevated access, and active character linkage.
- Account detail page with linked characters and administrative summary.
- Login metadata display with optional compatible login-history table support.
- Continued password hash/salt protection and read-only SQL enforcement.

No SQL migration is required.
