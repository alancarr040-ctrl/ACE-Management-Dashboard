# Release Notes - 2.9.0 Notifications & Alerts

2.9.0 introduces ACEMD's notification and alert foundation.

## Added

- Notification Service for local/dashboard alerts.
- Notifications route, API, and UI workspace.
- Active notification summary and navigation badge.
- Alert rules for Health, Metrics, Automation, and critical Events.
- Notification lifecycle actions for acknowledge and resolve.
- Local channel registry with deferred external channel placeholders.
- Notification lifecycle events in the operational event journal.

## Notes

External channels are intentionally deferred. This release builds the framework first so future delivery providers can be added safely.

## SQL

No SQL changes.
